Trader Performance vs Market Sentiment Analysis

This project analyzes how Bitcoin market sentiment (Fear vs Greed) affects trader behavior and profitability on the Hyperliquid platform.

The analysis includes:
- Data cleaning and merging
- Sentiment-based performance comparison
- Trader segmentation
- Behavioral insights
- Strategy recommendations

Tools Used:
Python, Pandas, Matplotlib, Seaborn, Google Colab

Key Insights:
Trading activity increases during Fear periods while profitability tends to improve during Greed market sentiment.
